# 🍗 Bar Manager API

API RESTful para gerenciamento de pedidos, estoque e caixa
de um bar/restaurante.

## 📋 Sobre o projeto

Sistema desenvolvido para digitalizar as operações de um bar,
substituindo o controle manual de pedidos, estoque e pagamentos.
Atende pedidos presenciais e entregas, com controle de acesso
por perfil (admin e atendente).

## 🚀 Tecnologias

- Java 17
- Spring Boot 3.5.14
- Spring Data JPA + Hibernate
- Spring Security + JWT
- PostgreSQL
- Flyway Migration
- Maven
- Lombok

## 📦 Módulos

- Autenticação e controle de acesso (JWT + Roles)
- Gestão de produtos e estoque
- Cadastro de clientes
- Registro de pedidos (presencial e entrega)
- Controle de caixa e formas de pagamento
